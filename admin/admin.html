📖 Songs Upload Guide

🎵 Method 1 — Google Drive (Free)

  1. drive.google.com లో MP3 upload చేయండి
  2. File → Share → Anyone with the link → Viewer
  3. Copy link → Admin panel లో paste చేయండి
  4. App లో automatically play అవుతుంది ✅

🌐 Method 2 — Direct MP3 URL

  1. ఏదైనా website నుండి direct .mp3 link వాడవచ్చు
  2. Example: https://example.com/song.mp3
  3. Link directly paste చేయండి

⚠️ Important Notes

  1. Google Drive share link ని app automatically convert చేస్తుంది
  2. Internet connection అవసరం songs play అవడానికి
  3. Large MP3 files slow internet లో buffer అవుతాయి
